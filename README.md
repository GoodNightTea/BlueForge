# BlueForge - BLE Security Research Framework

A modular framework for Bluetooth Low Energy security research and vulnerability analysis.

## ⚠️ RESEARCH USE ONLY

This framework is designed for security research on devices **you own**. 
Do not use on devices you do not have explicit permission to test.

## Features

- 🔍 **BLE Device Discovery**: Comprehensive scanning and enumeration
- 🧪 **Memory Pattern Analysis**: Systematic testing of memory corruption vectors  
- 📊 **Crash Detection**: Automated anomaly and crash detection
- 🎯 **Modular Exploits**: Extensible framework for security research
- 📝 **Research Logging**: Detailed logging for vulnerability analysis
